# 下载工具指南 - 只要这一份就够了

## [aria2c](https://aria2.github.io/)

### 用法示例

    # --enable-rpc=false: 禁用 rpc
    # -s: 单个任务对的最大线程数目
    # -x: 统一服务器的的连接数
    # -c: 启用断点续传功能
    # -o: 输出的文件名为 <file name>
    aria2c --enable-rpc=false -x 5 -s 5 -c <url> -o <file name>

## [wget](https://www.gnu.org/software/wget/)

    # 用户名密码下载 ftp 的某个目录
    # -r: 递归的 -np: 不向上下载父目录
    wget -r -np --user=<username> --password='<password>' ftp://ftp/dir/
